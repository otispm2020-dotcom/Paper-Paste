# Paper Paste

`Paper Paste` 是一个 Chrome 扩展，用来把当前浏览器标签页整理到同一张画布中，生成可搜索、可聚类、可编辑的网页卡片。

## 核心能力

- 同步全部标签页或当前窗口到画布
- 基于网页 HTML / 抓取文本生成摘要
- 支持导入链接生成卡片
- 支持删除、编辑卡片内容
- 支持按标题和摘要关键词搜索
- 支持按窗口、域名、内容聚类浏览
- 支持导出 PNG 和 JSON
- 支持配置多家模型服务：
  - SiliconFlow
  - Google Gemini
  - OpenRouter
  - OpenAI
  - Anthropic
  - 自定义 OpenAI 兼容接口

## 使用方式

1. 点击扩展图标打开 `Paper Paste`
2. 在“系统设置”中配置模型提供商、模型名称、接口地址和 API Key
3. 点击“同步全部标签页”或“仅同步当前窗口”生成卡片
4. 也可以直接输入链接，通过“导入链接”生成卡片
5. 使用顶部搜索框快速检索标题 / 摘要内容
6. 使用“聚类视图”切换不同聚合方式

如果未配置模型密钥，在执行导入或同步时会自动弹出设置窗口。

## 卡片内容

- 标题：网页标题
- 摘要：基于页面内容生成的中文摘要
- 链接：原始网页地址
- 快照：标签页截图

## 安装开发版

1. 打开 Chrome，进入 `chrome://extensions`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录

## 打包产物

仓库外可生成以下发布物：

- `.crx`：Chrome 扩展打包文件
- `.zip`：源码分发包
